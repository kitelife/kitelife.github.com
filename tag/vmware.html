    1. 实现strcpy。函数原型：char strcpy(char dest, const char* src)。

    后来发现自己实现的有问题。正确的实现应该如下：

    char* strcpy(char* dest, const char* src)
{
    assert(dest!=NULL && src!=NULL);
    char* ret = dest;
    while((*dest++ = *src++) != '\0');
    return ret;
}

    这个题目做得不好，我对自己很失望啊。
